# Container

`<Container>` is a helper component exported by this library. It comes in handy when you
want to demonstrate multiple components in `<Playground>`

## Usage

```js
import { Container } from 'docz-theme-ztopia/helpers';
```

## Demo

### Without Container

<Playground>
  <div
    style={{
      width: 200,
      height: 200,
      backgroundColor: 'red',
    }}
  />
  <div
    style={{
      width: 200,
      height: 200,
      backgroundColor: 'blue',
    }}
  />
  <div
    style={{
      width: 200,
      height: 200,
      backgroundColor: 'green',
    }}
  />
</Playground>

### With Container

<Playground>
  <Container centered style={{ height: 400 }}>
    <div
      style={{
        width: 200,
        height: 200,
        backgroundColor: 'red',
      }}
    />
    <div
      style={{
        width: 200,
        height: 200,
        backgroundColor: 'blue',
      }}
    />
    <div
      style={{
        width: 200,
        height: 200,
        backgroundColor: 'green',
      }}
    />
  </Container>
</Playground>

## API

| Property | Type          | Default | Required | Description                                                                            |
| -------- | ------------- | ------- | -------- | -------------------------------------------------------------------------------------- |
| centered | string        | `false` | false    | All child elements will be centered both horizontally and vertically in `<Playground>` |
| gap      | number        | `10`    | false    | Spacing between child elements                                                         |
| style    | CSSProperties | `{}`    | false    |                                                                                        |
